# Student Grade System

This project is a simple student grade tracking application that I developed while learning Python.  
It calculates, saves, lists, and allows searching student information based on user input.

---

## Project Features

- Add student  
- List students  
- Delete student  
- Update student  
- Find student by ID  
- Search by name, surname, or first letter  
- Calculate average  
- Assign letter grades (AA–FF)  
- Save data to a JSON file  
- Validate grade input (0–100 range)  

---

## Technologies Used

- Python  
- JSON
